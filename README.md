# Guardians of the Wild

**Guardians of the Wild** is a simple interactive wildlife-themed web game start page built using **HTML, CSS, and JavaScript**. The interface features a jungle background video, a nature-inspired design, and an interactive toggle that allows the player to begin the game.

When the user enables **“Protect the Jungle”**, the **Start Game** button appears. After clicking it, the player is asked to enter their name through a modal window. The name is stored using **localStorage**, and the player is then redirected to the **game menu page**.

## Features

* Full-screen **jungle background video**
* Dark overlay for better readability
* Interactive **toggle switch** to activate the game
* Animated **Start Game button**
* Player **name input modal**
* Name stored using **browser localStorage**
* Redirects to **menu.html** to continue the game

## Technologies Used

* HTML5
* CSS3
* JavaScript

  # Wild Game Zone – Menu Page

**Wild Game Zone** is the main menu page of the *Guardians of the Wild* web game. It allows players to choose from different wildlife and environment-themed mini games designed to create awareness about animals, conservation, and eco-friendly actions.

When the page loads, the player’s name (stored in **localStorage**) is displayed in a welcome message. The interface shows multiple **game zones**, each containing different jungle-themed mini games.

## Features

* Personalized **welcome message** using stored player name
* Animated **moving animal emojis** in the background
* Organized **game categories (zones)**
* Interactive **game buttons** linking to different mini games
* Clean jungle-themed layout using **HTML and CSS**

## Game Zones

* **Core Awareness Zone** – Wildlife knowledge and quizzes
* **Eco Action Zone** – Environmental protection activities
* **Animal Fun Zone** – Animal-based interactive games
* **Memory & Logic Zone** – Brain and memory challenges
* **Quick Play Zone** – Fast mini games
* **Story Adventure Zone** – Story-based conservation missions
* **Creative Corner** – Puzzles, sounds, and creative activities

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Browser **localStorage**
